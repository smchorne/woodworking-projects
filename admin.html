Deploying Changes

Changes saved here update your browser instantly. To make them permanent on the live site:

  1. Click Export JSON above to download the updated file.
  2. Replace wood-species.json in your repo with the downloaded file.
  3. Run git add wood-species.json && git commit -m "Update species data"
  4. Run git push origin main
